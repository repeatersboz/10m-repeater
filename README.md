# 10m repeater

Scripts for the 10 meter repeater

## Common:
- RPi up Led
- Reboot / Shutdown Led & Switch
- Case fan controller
- Temperature logger

## Transmitter:
- Amplifier fan controller
- TX Led
- RX UP Led

## Receiver:
- Squelch LED
- TX UP Led

## Remarks & notes
The case temperature is logged by the DHT22 module. This needs the wiringpi library en the lol_dht22 package.
